# **Killer Beats**

---

**Killer Beats** - ритм-игра, в которой нужно уворачиваться от препятствий под эпичную музыку. Уровни генерируются
рандомно, подстраиваясь под бит трека и увеличивающуюся с течением времени сложность.

---

Техническое задание:

✅ Создать классы игрока (Player) и препятствий (Slash, Bullet)

✅ С помощью библиотеки mido написать алгоритм извлечения ударных из трека

✅ Реализовать основные методы (setup, on_draw, on_update, on_key_press, on_mouse_press и т. п.)

✅ Добавить секундомер и подсчет очков, постепенное увеличение сложности

✅ Создать главное меню и эндскрин, добавить уровни

➡️ Проверить код на ошибки и баги
