
# **{violet}(Beat Slasher)**

---

**{violet}(Beat Slasher)** - ритм-игра, в которой нужно уворачиваться от препятствий под эпичную музыку. Уровни генерируются рандомно, подстраиваясь под бит трека и увеличивающуюся с течением времени сложность. В качестве препятствий выступают линии, круги, квадраты и тому подобные геометрические фигуры.  Также, имеется внутриигровой магазин со скинами и треками.
